## **3-Minute Setup (File Manager)**

1. **Log into Hostinger** → Files → File Manager
2. **Create folder structure:**
   - `css/` → Upload `style.min.css`, `animations.min.css`
   - `js/` → Upload `main.min.js`, `utils.min.js`, `animations.min.js`
   - `assets/` → Upload `images/`, `fonts/`, `icons/` folders
   - `case-studies/` → Upload all 6 HTML files

3. **Upload root files:**
   - index.html, privacy-policy.html, terms-of-service.html, cookie-policy.html
   - robots.txt, sitemap.xml

4. **Create .htaccess file:**
   - Right-click in public_html → New File → Name: `.htaccess`
   - Paste content from `DEPLOYMENT-HOSTINGER.md` file

5. **Test:** Open `https://yourdomain.com` in browser

## **Minimal .htaccess (If Full Version Too Complex)**

```apache
<IfModule mod_deflate.c>
    AddOutputFilterByType DEFLATE text/html text/css text/javascript application/javascript
</IfModule>

<IfModule mod_expires.c>
    ExpiresActive On
    ExpiresByType text/css "access plus 1 year"
    ExpiresByType application/javascript "access plus 1 year"
    ExpiresByType image/png "access plus 1 month"
    ExpiresByType image/jpeg "access plus 1 month"
    ExpiresByType image/svg+xml "access plus 1 month"
    ExpiresDefault "access plus 7 days"
</IfModule>
```

## **Troubleshooting**

| Problem | Solution |
|---------|----------|
| CSS not loading (page looks broken) | Check file paths in index.html match uploaded folder names |
| JavaScript errors in console | Clear browser cache (Ctrl+Shift+Delete) and hard refresh (Ctrl+F5) |
| Images broken (404 errors) | Verify /assets/images/ folder was uploaded with all files |
| Forms don't work | Check browser console (F12) - JavaScript should run without errors |
| Slow page load | Verify .min files being used; check .htaccess caching enabled |
| Case study pages return 404 | Confirm case-studies/ folder exists with all 6 HTML files |
